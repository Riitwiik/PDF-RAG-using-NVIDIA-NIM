# PDF-RAG-using-NVIDIA-NIM

📄 RAG using Nvidia NIM

A production-ready Retrieval-Augmented Generation (RAG) application built with Streamlit that allows users to upload PDFs and ask questions based on their content.

🚀 Features

📂 Upload multiple PDF documents

🔍 Intelligent document retrieval using FAISS

🤖 LLM-powered answers using NVIDIA NIM

⚡ Fast semantic search with embeddings

📊 Context-aware responses (no hallucination)

⏱ Response time tracking

🧠 Clean and modular architecture


🚀 Live Demo
